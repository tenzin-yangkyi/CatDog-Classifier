# 🐱🐶 Cats vs Dogs Image Classifier

This project is a Deep Learning-based web application that can identify whether an uploaded image is a cat or a dog. It was developed to explore the practical use of Convolutional Neural Networks (CNNs) in image classification and to gain hands-on experience with Computer Vision and Deep Learning.

The model was trained on thousands of cat and dog images, allowing it to learn important visual features such as shapes, fur patterns, facial structures, and other distinguishing characteristics. Once trained, the model can analyze a new image and predict whether it contains a cat or a dog.

To make the project easy to use, a simple web interface was created using Flask. Users can upload an image directly through the website and receive an instant prediction along with the model's confidence level.

This project helped in understanding the complete machine learning workflow, including data preprocessing, model building, training, evaluation, and deployment. It also demonstrates how AI models can be integrated into real-world applications through web technologies.

## Features

* Upload an image of a cat or dog
* Instant AI-powered prediction
* CNN-based image classification model
* User-friendly web interface
* Real-time results with confidence score

## Technologies Used

* Python
* TensorFlow & Keras
* Flask
* HTML, CSS, and JavaScript
* NumPy and Matplotlib
